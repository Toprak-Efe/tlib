# TLib++

This a personal template library that I intend to use in all my projects. It consists of `concepts` which are helpful templates to designate specific contracts & policies, `templates` that implement those concepts that other classes may choose to utilize--most of which are thread-safe and realtime-safe.

In some instances, these library features will augment existing types within the C++ STL, such as adding serialization. In other instances, the library will implement a simple physics system using libraries such as Eigen3 to complement my usual workflow of digital control systems implementations, it is a dump for all features that I might have found to be useful for my future projects.
